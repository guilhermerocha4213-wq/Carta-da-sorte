# Imagens das Cartas

Esta pasta deve conter as 52 imagens SVG das cartas de baralho.
As imagens devem ser nomeadas conforme o padrão esperado pelo `script.js` (ex: `ace_of_hearts.svg`, `2_of_hearts.svg`, etc.).

Você pode baixar SVGs gratuitos em: https://tekeye.uk/playing_cards/svg-playing-cards
